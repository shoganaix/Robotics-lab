# Robotics Lab

This repository contains a series of **robotics projects** that I have developed individually and as part of a team. 

The projects include mechanical design (3D modeling), electronics integration, and embedded programming. *All mechanical structures were designed by me*

---

## Projects

### 1. Spider Robot with Camera and Mobile Control
- 3D designed structure
- Integrated camera module
- Remote control via mobile application
- App developed using MIT App Inventor
- Wireless communication
- Embedded control system

Features:
- Real-time mobile control
- Live camera visualization
- Modular leg structure
- Custom chassis design

---

### 2. Logic-Gate Controlled Car
- Direction and steering system designed in 3D
- Chassis fully modeled by me
- Remote control operation
- Control logic implemented using digital logic gates

Features:
- Hardware-based logic control
- Custom steering mechanism
- Electronic circuit integration
- Mechanical + electronic system fusion

---

### 3. Sumo Robot with Suction System
- Custom 3D printed chassis
- Ground suction mechanism for increased traction
- Designed for sumo-style robotic competitions
- Autonomous combat-oriented structure

Features:
- Suction system to increase grip
- Optimized weight distribution
- Robust mechanical structure
- Embedded control electronics
